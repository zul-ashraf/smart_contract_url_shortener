## Running the website

```
git clone https://github.com/zul-ashraf/url_shortener.git
cd front_end
yarn
yarn start
```

## Example of registered URL:

```
http://localhost:3000/851 
//=>Redirect to (https://facebook.com)
http://localhost:3000/477 
//=>Redirect to (https://google.com)
```
